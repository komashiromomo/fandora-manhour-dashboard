# Fandora 人工時管理系統 Dashboard

員工工時追蹤與分析平台。

## Tech Stack

- React 18 + Vite 5
- Ant Design 5 + Recharts 2
- Google Identity Services (OAuth 2.0)
- SheetJS (xlsx)

## Setup

```bash
npm install
cp .env.example .env  # 填入 Google API credentials
npm run dev
```

## Deploy

Vercel auto-deploy on push to main.
